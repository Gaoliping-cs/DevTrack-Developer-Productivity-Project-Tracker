# DevTrack-Developer-Productivity-Project-Tracker


DevTrack is a full-stack web application that helps developers manage projects, track tasks, and monitor productivity.

Built with:

- React (Vite)
- Node.js
- Express
- MongoDB
- JWT Authentication
- Chart.js
- TailwindCSS
